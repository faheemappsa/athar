# Athar V2 Content Inventory

## Current confirmed catalog size

After the expanded packs, the unified V2 catalog contains 54 items.

Approximate distribution:

- Ayahs: 15+
- Duas: 12+
- Hadith: 10+
- Tafsir and meanings: 15+

## Important note

This is a stronger starter library, not the final massive religious content bank.

The final target remains a much larger offline-first DB generated from vetted source datasets through the build-time pipeline.

## Runtime

The live card can use this catalog only behind `VITE_ATHAR_V2=1`. The legacy engine remains fallback.
