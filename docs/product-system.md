# Athar Product System

Athar is a modern Arabic spiritual sharing web app.

Core journey:
1. User opens the app.
2. App shows current time and next prayer.
3. App presents a ready card with ayah, hadith, or dhikr.
4. User chooses destination size: WhatsApp, Snapchat, TikTok, square post.
5. User exports a polished image and shares it.

Core principles:
- Closed verified content library.
- No free religious text input in the first version.
- Editorial Arab visual identity.
- Premium backgrounds and controlled templates.
- Fast PWA web app.
